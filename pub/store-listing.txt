Full description:
This application adds RTM Terrebonne-Mascouche (MRC Les Moulins Urbis) buses information to MonTransit.

This app contains the buses schedule and the latest news from @exo_TMascouche on Twitter.

RTM Terrebonne-Mascouche (MRC Les Moulins Urbis) serves Mascouche and Terrebonne.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Réseau de Transport Métropolitain (RTM).
https://rtm.quebec/en/about/open-data

This application is free and open-source:
https://github.com/mtransitapps/ca-les-moulins-mrclm-bus-android

This app is not related with the Autorité Régionale de Transport Métropolitain (ARTM) and the Réseau de Transport Métropolitain (RTM) Terrebonne-Mascouche (MRC Les Moulins Urbis).
